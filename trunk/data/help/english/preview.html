Preview Panel

Review panel in the vertical orientation of the results is located in the left side of the main window:

or at the top of the main window when horizontal orientation:

Switching the orientation is carried out using the item View - Orientation of Main Menu. The functionality of the preview pane does not depend on the orientation and is designed to display the current result of Table of Results. With it you can examine in detail the duplicate or defective images, as well as make the basic operations on them.

Showing of pair of duplicates

A pair of duplicate images is displayed on the panel in a column or row (depending form orientation). The relative position of images corresponds to their position in the list of results. The control buttons are placed near them. Next is a list of these buttons and their names:

  • or - Delete both pictures.

  • or - Replace second picture by the first picture.

  • or - Delete first picture.

  • - Go to the previous result.

  • - Go to the next result.

  • or - Delete second picture.

  • or - Replace first picture by the second picture.

  • - Mark current result as a mistake.

The meanings of most actions undertaken by the buttons are well understood from their names. We describe only the last: this button allows you to mark the current result as false that would exclude his appearance after repeated searches. Additionally, we note that all the above actions (except for moving between the results) may be undone.

howing of defect picture

Defective image is displayed at the top panel or at the left panel (depending form orientation). It might happen that the program can not open it and the top panel will be a blank image. There are control buttons near:

  • or - Delete picture.

  • - Go to the previous result.

  • - Go to the next result.

  • - Mark current result as a mistake.

Additional features

  • Under each picture is the panel with some information about it. It includes the path to the image, its type, the size (width and height) and its file size on the disk. Note that if a duplicate image pairs have a different type, then this information is highlighted in red. Also highlighted in red image information having a smaller dimensions or smaller file size. Double-clicking on the name of the picture, you can rename or move a given image.

  • When you double-click the mouse on the picture, it will be opened with using the external picture viewer.

  • In some cases, when the program can uniquely identify a better picture of the pair, it will create a recommendation for the user, and the corresponding button will be marked in pink.

  • All buttons on this panel are duplicated by the Hot Keys. Current hot keys appear in the description of the buttons in tooltips, if you move the mouse cursor to any of the buttons on the panel.

  • If you right click on the picture, it opens a context menu where you can:

    • Open the picture with using an external picture viewer.

    • View the directory, which contains the current image.

    • Copy the path to this picture.

    • Rename or move current image.

    • Move image into the directory of neighboring image. This operation is enabled if these images are placed in different directories and have a different names.

    • Note that the move operation or rename the images are undone.
