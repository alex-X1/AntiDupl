Введение

У современных пользователей на компьютере обычно хранятся большие коллекции различных изображений. Если объем коллекции превышает несколько тысяч картинок, то, как правило, при её обновлении и хранении встаёт вопрос наличия большого количества дублирующих изображений. Находить и удалять их вручную - занятие весьма утомительное. В этом вам поможет программа "AntiDupl". Она способна находить и выводить дубликаты изображений в основных графических форматах: JPEG, GIF, TIFF, BMP и PNG. Сравнение ведётся по содержимому файлов, поэтому, в зависимости от настроек программы, программа может находить не только полностью одинаковые изображения, но и похожие картинки, т.е. одно и то же изображение может содержаться в файлах с разным форматом, степенью сжатия и разрешением. Кроме того, программа умеет находить изображения с определёнными дефектами.
Несомненно, у "AntiDupl" есть аналоги, однако в пользу "AntiDupl" говорят простота в управлении, высокая скорость и точность работы, поддержка русского и английского интерфейса, бесплатность.

Установка

Программа "AntiDupl" не требует установки.

Системные требования

Операционная система Windows XP/.NET Server/Vista.
Для того чтобы программа выполнялась на предыдущих версиях Windows (Windows 98/ME, Windows NT SP6 или Windows 2000), необходимо добавить в системный каталог файл gdiplus.dll. Найти его можно, например, здесь.

После чего программа начнёт в заданной папке поиск одинаковых или повреждённых картинок. В случае нахождения таковых картинок программа выдаст диалог с запросом. В главном окне программы отображается её состояние и результаты работы. Обратите внимание, что результаты поиска сильно зависят от его параметров.
Остановить поиск можно, выбрав пункт меню Стоп (Stop), либо нажав кнопку

на панели инструментов, либо при помощи горячих клавиш.

Диалог сравнения и удаления

В случае нахождения одинаковых картинок программа выдаст диалог с запросом:

В этом диалоге отображаются сами картинки, их название и путь относительно каталога поиска (полный путь показывается при наведении мыши), их ширина и высота, размер занимаемых ими файлов (меньшие параметры выделяется красным цветом). Вы можете удалить одну из этих картинок, нажав кнопку

под ней, либо заместить соседней картинкой, нажав кнопку

Если вы хотите оставить/удалить обе картинки, то воспользуйтесь кнопками Пропустить (Next)/ Удалить обе (Delete both). При нажатии кнопки Стоп программа остановит поиск. Если убрать галочку в опции Запросы во время поиска (Queries during search), то программа не будет больше выводить данный диалог во время поиска (диалог будет показан только по его окончании). Размер выводимых картинок можно изменять, растягивая диалог мышью. При двойном щелчке по картинке она будет открыта в приложении, ассоциированном с этим типом файлов. При щелчке правой кнопкой мыши по картинке, появится всплывающее меню, в котором можно

  1. Скопировать путь (Copy path) к картинке в буфер обмена.
  2. Открыть картинку (Open picture) в приложении, ассоциированном с этим типом файлов.

Начиная с версии 2.4 в диалоге появилась кнопка Ошибка (Mistake). С помощью этой клавиши можно помечать ошибочные срабатывания программы. Сообщения пользователя об ошибках будут занесены в специальную базу данных и при повторных поисках данные пары картинок уже не будут выводится (аналогично не будут выводится картинки, которые программа ошибочно восприняла как дефектные). Данная кнопка видна, только если включен соответствующий режим в опциях программы.

Обратите внимание, что в диалоге действуют горячие клавиши.

P.S. При нахождении программой повреждённой картинки будет показан аналогичный диалог.

Отображение состояния и результатов работы программы

Во время программы существует три основных состояния: ожидание нового поиска (зелёный значок), процесс поиска (жёлтый значок) и показ результатов, т.е. вывод диалога сравнения и удаления картинок (красный значок). Ход поиска отображается в процентах в заголовке окна программы (а также на индикаторе внизу главного окна). В строке состояния поясняются текущие действия программы. За отображение строки состояния отвечаетт пункт меню Вид - Показывать строку состояния (View - Show status bar).
Кроме того, в главном окне программы ведётся отчёт поиска, в котором отображаются основные его результаты:

  • Время запуска и папка поиска.
  • Количество найденных вложенных папок и содержащихся в них графических файлов.
  • Количество картинок, которые были проверены.
  • Количество найденных повреждённых или одинаковых картинок.
  • Количество удалённых картинок и занимаемый ими объём.

Отчёт поиска сохраняется в файле "AntiDupl.log", расположенном в папке размещения программы. Отчёт можно очистить, выбрав пункт меню Инструменты - Очистить отчёт (Tools - Clear log), или аналогичный пункт во всплывающем меню, вызываемом щелчком правой кнопки мыши по окну отчёта.

Обзор меню и панели инструментов

Меню программы состоит из следующих пунктов:

  • Файл (File)
    • Старт (Start) - данный пункт запускает поиск.
    • Обзор (Browse) - вызывает диалог выбора папки поиска.
    • Выход (Exit) - завершает исполнение программы.
  • Вид (View)
    • Показывать панель инструментов (Show toolbar) - вкл./выкл. отображение панели инструментов.
    • Показывать строку состояния (Show status bar) - вкл./выкл. отображение панели состояния.
  • Сервис (Tools)
    • Настройки (Options) - вызывает диалог с настройками программы.
    • Очистить отчёт (Clear log) - позволяет очистить отчёт программы.
  • Справка (Help)
    • Справка (Help) - вызывает справку.
    • О программе... (About AntiDupl...) - показывает диалог "О программе".
  • Стоп (Stop) - позволяет прервать поиск (появляется только во время поиска).

Панель инструментов предоставляет быстрый доступ к основным пунктам меню. При желании панель инструментов можно отключить (см. выше обзор меню).

  • - начало поиска.
  • - остановка поиска.
  • - выбор папки.
  • - вызов диалога настроек.
  • - вызов справки.

Обзор настроек

Диалог с настройками вызывается выбором пункта менюСервис - Настройки (Tools - Options) или кнопкой

на панели инструментов. Диалог настроек представляет собой многостраничное окно. Ниже приведено описание этих страниц:

  • Общие (General) - страница содержит настройки общего характера.

    • Поле Язык (Language) позволяет выбрать язык интерфейса программы: Русский (Russian) или Английский (English).

    • Искать в подпапках (Search in subfolders) - определяет, будет ли программа выполнять поиск только в выбранной папке, или также во всех вложенных в неё подпапках.

    • Удалять в корзину (Delete to recycle bin) - при включении этого параметра программа удаляет файлы в корзину, иначе она их будет удалять безвозвратно.

    • Запросы во время поиска (Queries during search) - данный параметр определяет, следует ли программе сразу выводить диалог с найденными картинками, или нужно подождать окончания поиска. Этот параметр можно также изменить в диалоге сравнения и удаления картинок.

    • На панели Ошибки (Mistakes) содержится опция Запоминать ошибки (Remember mistakes), которая включает режим запоминания ошибок. В этом режиме, как уже было сказано выше, программа заносит в базу данных свои ошибочные срабатывания, что позволяет ей не выводить их в дальнейшем. Кроме того, на этой панели есть кнопка Очистить (Clear) с помощью которой можно очистить базу данных ошибок.

  • Сравнение (Comparison) - страница содержит основные параметры сравнения картинок.

    • Искать похожие картинки (Check pictures on equality) - определяет, следует ли программе искать дубликаты изображений. При её отключении все остальные опции на данной странице будут недоступны.

    • Степень различия картинок (Degree of difference) - определяет как сильно могут различаться искомые изображения. У этого параметра всего пять возможных вариантов: Нет (None) - будут искаться только полностью совпадающие изображения, Маленькая (Small), Средняя (Normal), Большая (Big), Наибольшая (Very big) - будут искаться все более отличающиеся картинки. Обратите внимание, что при малой степени различия программа может не обнаружить некоторые дубликаты, а при большой увеличивается вероятность ложных срабатываний. Рекомендуется использовать значение Средняя (Normal).

    • Учитывать размер картинок (Consider picture sizes) - определяет, следует ли программе учитывать размер картинок. Например, при включении этого параметра картинки размерами 1024x768 и 800x600 будут считаться разными, даже если на них будет изображено одно и то же.

    • Учитывать формат файлов (Consider file formats) - определяет, следует ли программе учитывать формат изображений. При включении этого параметра, например, картинки в форматах PNG и GIF будут считаться разными, даже если на них будет изображено одно и то же.

    • Сравнивать картинки целиком (Compare full pictures) - этот параметр определяет, как будут сравниваться картинки - целиком или как их уменьшенные изображения. Включение данного параметра повышает точность предсказания, но замедляет работу.

    • Автоудаление одинаковых картинок (Auto delete equal pictures) - если включить этот параметр, программа будет удалять одинаковые картинки автоматически, без вывода запроса пользователю. Отметим, что программа удаляет автоматически только полностью одинаковые изображения.

  • Другие настройки (Other) - эта страница содержит все остальные настройки.

    • На панели Формат картинок (Picture format) можно выбрать форматы картинок, которые следует искать программе.

    • Проверять картинки на дефекты (Check picture integrity) - этот параметр определяет, следует ли программе искать повреждённые картинки. Программа может находить не полностью скачанные картинки в формате JPEG. Другие типы дефектов программа пока не находит.

    • Автоудаление повреждённых картинок (Auto delete defect pictures) - при включении этого параметра программа будет удалять дефектные картинки автоматически, без вывода запроса пользователю.

Горячие клавиши

Во время поиска действуют следующие "горячие" клавиши:

  • Pause/Break - при нажатии этой клавиши поиск будет остановлен (аналогично нажатию кнопки Стоп (Stop)). Кнопка также действует в диалоге сравнения и удаления картинок.

  • Shift - аналогично нажатию клавиши Пропустить (Next) в диалогах сравнения и удаления картинок.

  • Delete - удаляет левую картинку в диалогах сравнения и удаления картинок.

  • End - удаляет правую картинку в диалоге сравнения картинок.

  • Insert - помещает правую картинку на место левой в диалоге сравнения картинок.

  • Home - помещает левую картинку на место правой в диалоге сравнения картинок.

  • Page Down - удаляет обе картинки в диалоге сравнения картинок.

  • Page Up - заносит ошибку в базу данных в диалогах сравнения и удаления картинок.
